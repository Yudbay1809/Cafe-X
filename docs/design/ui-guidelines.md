# UI Design Guidelines

Style: modern coffee brand, clean minimal, rounded corners, soft shadows

Colors:
- Primary #6B4F3A
- Secondary #C8A27A
- Background #F8F5F2

Typography:
- Heading: Inter Bold
- Body: Inter Regular

Admin dashboard widgets:
- Revenue today
- Orders today
- Top products
- Sales chart

POS layout:
- Left category panel
- Center product grid
- Right cart
- Large touch-friendly buttons

Customer app:
- Mobile-first
- Menu list
- Product detail
- Add to cart
- Checkout
