# Cafe-X Pricing One-Pager

## Positioning
Cafe-X adalah platform operasional cafe untuk order meja QR, POS kasir, admin operasional, dan billing dalam satu stack.

## Paket
### Basic
- 1 outlet
- QR menu + order status
- POS kasir core (order, payment, receipt)
- Laporan dasar harian/shift
- Harga: Rp 299.000 / bulan

### Pro
- Semua fitur Basic
- Multi user + audit log lanjutan
- Sinkronisasi lebih advanced
- Billing dan onboarding tools penuh
- Harga: Rp 599.000 / bulan

### Premium
- Semua fitur Pro
- Multi outlet analytics
- Integrasi pihak ketiga (bertahap)
- Prioritas support
- Harga: Rp 1.199.000 / bulan

## Add-on Ops
