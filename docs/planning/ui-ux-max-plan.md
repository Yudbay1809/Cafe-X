# UI/UX Max Remodel Plan

Tanggal: 2026-04-07
Scope: customer-next, admin-next, pos-flutter
Tujuan: premium look, fokus ke kecepatan transaksi dan kejelasan informasi.

## Phase 0 - Audit & Target
- Tentukan app prioritas (customer/admin/pos).
- Kunci brand direction (tone, warna, font, feel).
- Kunci 3 metrik UX: kecepatan aksi, fokus visual, kontras.

## Phase 1 - Design System
- Tokens: colors, type scale, spacing, radius, shadow, motion.
- Typography: display + body.
- Grid/layout: desktop/tablet/mobile.

## Phase 2 - Key Screens Remodel
- Customer: menu + cart drawer + order status.
- Admin: dashboard + products + tables + QR print.
- POS: order grid + cart + payment flow.

## Phase 3 - Interaction & Motion
- Loading state halus + CTA hover.
- Animasi status transaksi (pending/success/failed).
- Micro-interaction cart add/remove.

## Phase 4 - Consistency & Polish
- Spacing rhythm 8/12/16/24.
- Harmonisasi icon/button/badge/input.
- Polish small screen.

## Phase 5 - QA Visual
- Contrast, font scaling, tap targets.
- Checklist per page (readability/hierarchy/focus/empty state).
- Final pass: performance + bundle size.

## Deliverables
- UI tokens in code (CSS/Theme).
- Updated key screens per app.
- Motion guidelines & loading states.
- QA checklist results.
