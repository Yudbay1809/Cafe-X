NEXTJS SHADCN UI SETUP STRUCTURE

GOAL

create modern admin dashboard
use shadcn ui
use scalable feature architecture

STACK

nextjs app router
tailwind
shadcn ui
react query
zustand
react hook form
zod

INSTALLATION STEPS

install nextjs latest

npx create-next-app@latest admin-next

install tailwind

install shadcn ui

npx shadcn-ui@latest init

install dependencies

npm install @tanstack/react-query
npm install zustand
npm install react-hook-form
npm install zod
